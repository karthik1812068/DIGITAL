Client testimonial videos
--------------------------
Copy your MP4 files from your WhatsApp export folder into this directory and name them:

  video-1.mp4
  video-2.mp4
  video-3.mp4
  video-4.mp4
  video-5.mp4

You can use fewer files — missing slots show a short hint until you add the file.
To use different filenames, edit src/data/clientVideos.js.
